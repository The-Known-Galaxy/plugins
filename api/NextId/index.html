NextId

Provides a way to create a callback that always returns a Unique Identifier, or UID for short, in the form of an infinitely incrementing natural number. Inspired by the brilliant justification from here (CTRL+F Don't type layout orders by hand). Thanks Kampfkarren!

NextId.New() → () → number

Returns a callback that can be used to freely create a new, unique identification number by invoking it.

local nextUniqueId = NextId.New()

print(nextUniqueId()) --> 1
print(nextUniqueId()) --> 2
print(nextUniqueId()) --> 3
